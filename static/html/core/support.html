Support

For Reforger Mods API support, billing questions, API key help, privacy requests, or abuse reports, email support@cedarline.digital.

What To Include

Include the request URL, approximate time of the issue, response status, request ID if one was returned, and the API key prefix if the issue involves authenticated access. Do not send full API keys, passwords, payment card numbers, private server credentials, or other sensitive secrets.

API Keys And Billing

For API key or paid-plan issues, include the account email, organization name if applicable, plan or invoice reference, and a short description of what changed or stopped working.

Abuse And Security

For abuse, scraping, exposed API keys, or security concerns, include enough detail to reproduce or verify the issue. We may rotate or revoke exposed keys and may limit traffic that threatens service reliability.

Privacy Requests

For privacy questions or account data requests, contact support from the email address associated with the API key or billing account where possible. Some operational, billing, security, tax, or legal records may need to be retained.

Service Status

If an endpoint returns 202 Accepted, the API has queued a background refresh job. Wait for the Retry-After value and retry the same URL before opening a support request.
