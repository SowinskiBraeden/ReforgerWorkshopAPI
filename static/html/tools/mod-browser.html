Arma Reforger Mods Browser

Search indexed Arma Reforger Workshop mod data, inspect Workshop IDs, versions, dependencies, scenarios, and add mods directly to a working server config.

What you can do here

Search by name or exact Workshop ID, copy IDs or config.json entries, add results to the config builder, and open detail pages for dependencies, scenarios, versions, and official Workshop links.

Where the data comes from

Results are cached public Workshop metadata normalized by the Reforger Mods API. First-time searches can still use the background refresh flow while cached data remains responsive.

Category

How to find an Arma Reforger mod ID

Search by mod name, author, category, or paste a 16-character Workshop ID. Each result shows copy actions for the ID and for a ready-to-use game.mods JSON entry. Detail pages also show the official Workshop link so you can verify the source before deploying.

How to add a mod to config.json

Use Add to Config from a result or detail page to send the mod into the shared browser draft used by the Arma Reforger Config Generator and Arma Reforger Mod Manager. You can also copy the JSON snippet and paste it into game.mods by hand.

How dependencies work

When Workshop exposes dependency data, the mod detail page lists the required mods. Dependency information can lag behind the official Workshop cache, so review the list before restart and validate the finished file with the Config.json Validator.

Popular mod categories

Category filters help narrow common server-admin searches: gameplay changes, weapons, vehicles, terrains, equipment, factions, scenarios, AI, audio, and miscellaneous tools. These are search filters, not rankings or endorsements.

Related tools

Building a server config? Open the Arma Reforger Server Config Generator. Only editing the mod list? Use the Arma Reforger Mod Manager. Building an integration? Start with the Arma Reforger Mods API.
