## Service Description

Reforger Mods API provides read-only JSON access to normalized metadata from publicly accessible Arma Reforger Workshop pages. Workshop data may include mod names, authors, descriptions, images, tags, ratings, download counts, update dates, dependencies, scenarios, and links back to official Workshop pages.

Workshop data is controlled by Bohemia Interactive and may change, disappear, become unavailable, or contain errors. Reforger Mods API caches and formats that public data but does not verify ownership, entitlement, moderation status, identity, legal rights, or platform account information.
