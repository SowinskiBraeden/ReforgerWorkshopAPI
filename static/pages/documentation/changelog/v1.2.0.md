# v1.2.0 - Server Admin Tools

<sup>*Released: 2026-07-09*</sup>

## Added

* Public Arma Reforger mod browser at [/arma-reforger-mods/](/arma-reforger-mods/), powered by the public API, with search, sorting, pagination, mod detail pages, copy-mod-ID and copy-JSON actions, and correct handling of `202 Accepted` refreshes.
* Config.json tools for server admins: a [config validator](/config-validator/) with local, in-browser JSON and structure checks, a [config generator](/config-generator/) with a form editor, live JSON preview, import/export, and a [mod manager](/mod-manager/) for the `game.mods` array with name resolution, reordering, duplicate detection, and dependency suggestions.
* Shared working config across tools: Add to Config in the mod browser, the mod manager, and the config generator all edit the same draft, stored in the browser only.
* [API quickstart page](/api/) and new guides: [config.json explained](/guides/arma-reforger-config-json/), [how to add mods](/guides/how-to-add-mods/), [config.json troubleshooting](/guides/config-json-troubleshooting/), [running server mods](/guides/arma-reforger-server-mods/), [API integration](/guides/api-integration/), and [handling 202 and refresh jobs](/guides/handling-202-refresh-jobs/).
* API documentation now includes client identity guidance (`User-Agent` / `X-API-Client`), conditional request examples, and complete code examples in curl, JavaScript, and Go.
* SEO metadata for all new pages: unique titles and descriptions, canonical URLs, Open Graph tags, structured data (SoftwareApplication, FAQPage, BreadcrumbList), and sitemap entries.
* Inline Workshop mod search inside the config generator and mod manager: search by name or paste a mod ID and add mods without leaving the editor.
* JSON syntax highlighting in the config validator editor and the generator live preview.
* Copied and exported mod entries include the mod `version`, resolved from Workshop data.

## Changed

* Homepage now presents both the API and the server-admin tools.
* Site navigation includes the tools and guides sections.
* Tool pages (mod browser, config generator, validator, mod manager, mod details) use a full-width desktop layout with refreshed colors and spacing.

No API endpoint, response-schema, or integration changes. The deprecated unversioned API aliases remain untouched.
