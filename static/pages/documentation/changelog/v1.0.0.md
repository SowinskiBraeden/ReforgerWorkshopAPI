# v1.0.0
<sup>*Released: 2026-07-04*</sup>

Initial public release of reforgermods.net and the Reforger Mods API.

## Included

- Read-only `/v1` API routes for health checks, mod lists, search, and mod details.
- Cached metadata responses sourced from publicly accessible Arma Reforger Workshop pages.
- Anonymous public rate limiting with proxy-safe client IP handling.
- Stale cache fallback for safe read endpoints when upstream Workshop pages are slow or unavailable.
- Controlled JSON error envelopes.
- Docker and reverse-proxy deployment documentation.
